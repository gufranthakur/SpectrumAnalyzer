#   Spectrum Analyzer 

<img width="1913" height="1076" alt="Screenshot from 2025-07-23 16-25-37" src="https://github.com/user-attachments/assets/832f8f63-3dbe-4bc3-9fe1-e2cba9dad727" />

A minimal signal processing and filtering application written in JavaFX.

## Features 
- Apply filters to signals (Butterworth, Chebishev)
- Modify ripple factor or order level
- Import and Export .WAV files (export feature under work)
- Calculate statistics for signals

## Technical Specifications 
- Programming language : Java 21
- GUI Framework : JavaFX
- Signal proccessing library : JDSP 
- Build tool : Maven
- Development and tested on : Ubuntu 24 (X11)
- Theme : Cupertino Dark (AtlantaFX)

## To run on your machine 
- Copy the git link :- ```https://github.com/gufranthakur/SpectrumAnalyzer.git``` 
- Open IntelliJ IDEA
- Close project (to go on home screen)
- Click on clone repository, paste the link
- Create and run the project (Main class : ```SpectrumAnalyzer.java``` 
You can use your preferred IDE, but IntelliJ is heavily recommended.
